# **App Name**: Fretum-Freight TMS

## Core Features:

- Authentication: User authentication with email/password and Google login using Firebase Auth.
- Dashboard: Display key metrics like 'Active Loads', 'Available Trucks', and 'Revenue This Month'.
- Load Management: List and manage loads with filtering and searching. New load modal with Google Places Autocomplete and Distance Matrix API integration.
- Dispatch Board: Drag-and-drop interface to assign loads to drivers. Displays driver availability.
- Document Parsing: Tool to automatically extract shipper, rate, and address details from uploaded rate confirmation PDFs using a Generative AI model to create draft loads.
- Invoicing: Displays invoices with filtering based on parameters, showing draft, sent and paid invoices.
- Fleet & Driver Management: Manage drivers and vehicles. Option to upload compliance documents with expiration date tracking.
- Customer Management: CRM for shippers and brokers with contact history and lane analytics.
- Integrations Hub: Connect with Gmail, load boards, ELD providers, and accounting software.
- Reports & Analytics: Revenue analysis, utilization rates, and custom report builder.

## Style Guidelines:

- Primary color: Professional Blue (#0066CC) for main interactive elements, navigation, and brand identity.
- Secondary color: Accent Green (#10B981) for success states, positive metrics, and call-to-action buttons.
- Background color: Clean white (#FFFFFF) with subtle blue tints (#F8FAFC) for a professional, enterprise look.
- Sidebar: Navy blue (#1E3A5F) for a sleek, professional navigation experience.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy feel. Body font: 'Inter' (sans-serif) for readability.
- Use modern, minimalist icons (Lucide) to represent various actions and modules, enhancing the user experience.
- Responsive design with collapsible sidebar navigation organized by categories (Main, Operations, System).
- Subtle transitions and animations to provide visual feedback and improve user engagement.
- Enterprise-grade UI with card hover effects, status indicators, and gradient accents.
- Accessibility: WCAG 2.1 AA compliant with proper focus states and color contrast.
