# **App Name**: FocusFreight TMS

## Core Features:

- Authentication: User authentication with email/password and Google login using Firebase Auth.
- Dashboard: Display key metrics like 'Active Loads', 'Available Trucks', and 'Revenue This Month'.
- Load Management: List and manage loads with filtering and searching. New load modal with Google Places Autocomplete and Distance Matrix API integration.
- Dispatch Board: Drag-and-drop interface to assign loads to drivers. Displays driver availability.
- Document Parsing: Tool to automatically extract shipper, rate, and address details from uploaded rate confirmation PDFs using a Generative AI model to create draft loads.
- Invoicing: Displays invoices with filtering based on parameters, showing draft, sent and paid invoices.
- Fleet & Driver Management: Manage drivers and vehicles. Option to upload compliance documents with expiration date tracking.

## Style Guidelines:

- Primary color: Vibrant green (#10B981) for main interactive elements and the brand's primary message.
- Background color: Very light green (#F0FAF7) to create a bright, clean, professional background.
- Accent color: Calming blue (#3B82F6) for secondary interactive elements, bringing a sense of reliability.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy feel. Body font: 'Inter' (sans-serif) for readability.
- Use modern, minimalist icons to represent various actions and modules, enhancing the user experience.
- Responsive design with sidebar navigation for desktop and collapsible menu for mobile.
- Subtle transitions and animations to provide visual feedback and improve user engagement.
